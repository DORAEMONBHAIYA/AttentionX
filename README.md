# 🚀 AttentionX

> Transforming long-form content into powerful short-form insights using AI

---

## 📌 Overview

**AttentionX** is an AI-powered content repurposing engine designed to extract high-value insights ("golden nuggets") from long-form content like lectures, podcasts, and workshops.

In a world where attention spans are shrinking, AttentionX helps creators convert hours of content into engaging short-form clips optimized for modern platforms.

---

## 🎯 Problem Statement

Creators produce valuable long-form content, but:

* ⏳ Important insights are buried in lengthy videos
* 📉 Audience prefers short, bite-sized content
* 🧠 Manual clipping is time-consuming and inefficient

---

## 💡 Solution

AttentionX automatically:

* 🎥 Processes long videos
* 🔍 Identifies high-impact moments
* ✂️ Extracts short, engaging clips
* 📱 Optimizes content for reels, shorts, and social media

---

## ⚙️ Features

* 🔎 **AI Highlight Detection**
  Detects the most valuable and engaging segments

* 🧠 **Smart Content Understanding**
  Uses NLP/LLMs to analyze context and meaning

* 🎬 **Auto Clip Generation**
  Converts long videos into short-form content

* 📊 **Engagement Optimization**
  Focuses on viral and impactful moments

---

## 🛠️ Tech Stack

* 🧠 AI / ML Models (NLP, Video Processing)
* 🎥 Video Processing Tools (FFmpeg / OpenCV)
* ⚡ Backend (Python / FastAPI)
* 🌐 Frontend (React + Tailwind CSS)
* ☁️ Database / Storage (MongoDB / Cloud)

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/DORAEMONBHAIYA/AttentionX.git
cd AttentionX
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the project

```bash
# backend
python app.py

# frontend
npm install
npm start
```
